Este proyecto consiste en una página web sencilla de una sola interfaz desarrollada como práctica de la materia Aplicaciones Web Orientadas a Servicios.
La página tiene como temática el Fútbol Club Barcelona, elegida libremente para la realización del ejercicio.

La aplicación está construida utilizando HTML, CSS y JavaScript, integrando contenido informativo, imágenes y animaciones. En el pie de página, al hacer clic sobre el texto “FC Barcelona”, se despliega una animación interactiva que muestra un video, cumpliendo con los requisitos de animación solicitados en la práctica.

El objetivo principal del proyecto es aplicar los conocimientos básicos de desarrollo web, estructura de una página, estilos visuales, uso de JavaScript para interactividad y publicación del sitio mediante GitHub Pages.
